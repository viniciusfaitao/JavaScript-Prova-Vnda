## Compilar SASS

  ```sh
  Abra um terminal e vá até a pasta src
  
  cd src

  após instale o gulp e o gulp-sass

  npm i gulp
  npm i gulp-sass

  finalmente, para compilar o SASS em CSS

  gulp sass
  
  ```
